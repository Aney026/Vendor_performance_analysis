Vendor Performance Analysis – README
Overview

This project analyzes vendor performance using sales, profit, and operational metrics. The goal is to identify high-performing vendors, uncover inefficiencies, and support data-driven decision-making in procurement and supply-chain management.

Objectives

Evaluate vendor sales, gross profit, and profit margins

Identify top- and low-performing vendors

Detect trends and inconsistencies in vendor operations

Provide insights to improve vendor selection and negotiations

Key Features

Data cleaning and preprocessing

Exploratory Data Analysis (EDA) with charts

KPI computation (Sales, Profit, Margin, Quantity)

Vendor ranking and segmentation

Business problem identification

Tools Used

Python

Pandas – data manipulation

NumPy – numerical operations

Matplotlib / Seaborn – data visualization

Jupyter Notebook – analysis and reporting

How to Use

Load the vendor dataset (CSV/Excel).

Run the notebook or script to clean and process the data.

Review summary tables and visual insights.

Use identified patterns to support vendor-related decisions.

Business Value

Improved vendor negotiation strategy

Optimized sourcing and cost control

Reduced operational risks

Higher supply-chain efficiency
